# 🧠 AI Grammar Genius

AI Grammar Genius is a modern AI-powered grammar correction web application built using Node.js, Express.js, EJS, and OpenRouter API.

It helps users instantly improve grammar, spelling, punctuation, and sentence clarity using artificial intelligence.

---

# 🚀 Features

- ✨ AI-powered grammar correction
- ⚡ Fast text processing
- 🧠 Smart sentence understanding
- 📋 Copy corrected text
- 🔄 Loading/typing indicator
- 🎨 Modern responsive UI
- 🔒 Secure environment variables
- 📱 Mobile-friendly design

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-grammar-genius.git
```

## 2️⃣ Open Project Folder

```bash
cd ai-grammar-genius
```

## 3️⃣ Install Dependencies

```bash
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```env
OPENROUTER_API_KEY=your_openrouter_api_key
SESSION_SECRET=your_secret_key
```

---

# ▶️ Run Project

```bash
npm start
```

OR

```bash
node app.js
```
